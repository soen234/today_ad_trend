Complete Guide to Mobile Banner Ad Sizes: Which Size Works Best?

320x50, 300x250, 320x100... Which banner size has the highest CTR and eCPM? We analyze based on real data.

Why Banner Ad Size Matters

Banner ad size isn't just about pixels. Size significantly affects user experience, click-through rate (CTR), and revenue (eCPM). Choosing the right size can more than double your ad revenue.

Introduction to Main Banner Sizes

1. Standard Banner (320x50)

The most basic and widely used mobile banner size. It stays fixed at the bottom of the screen for continuous exposure.

320 x 50 pixels
Standard Banner

Pros

  • Minimal screen space, minimal user disruption
  • High fill rate across almost all ad networks
  • Can be displayed continuously throughout app

Cons

  • Relatively low eCPM ($0.5-$2)
  • Low CTR due to small size
  • Limited creative expression

2. Large Banner (320x100)

Twice the height of Standard Banner, providing greater visibility and higher CTR.

320 x 100 pixels
Large Banner

üí° Recommended Use

When there's room at the bottom of the screen, you can expect 20-30% higher eCPM than Standard Banner.

3. Medium Rectangle (300x250)

The banner size with the highest eCPM. It's inserted between content for high visibility.

300 x 250 pixels
Medium Rectangle

Pros

  • Highest eCPM among banners ($2-$5)
  • High fill rate due to advertiser preference
  • Rich creative expression possible

Cons

  • Large screen area requires careful positioning
  • Only suitable for scrollable feeds
  • Overuse may cause user churn
Advertisement

4. Adaptive Banner (Dynamic Size)

A smart banner that automatically determines the optimal size based on device screen width. This is strongly recommended by Google AdMob.

üéØ 2025 Recommendation

Adaptive Banner displays at optimal size across devices, generating 20% higher revenue on average. Use Adaptive Banner as default for new projects.

Performance Comparison by Banner Size

Size Avg eCPM CTR Fill Rate Rating
320x50 $0.5-2 0.1-0.3% 95%+ Basic
320x100 $1-3 0.2-0.4% 90%+ Good
300x250 $2-5 0.3-0.8% 95%+ Best
Adaptive $1-4 0.2-0.5% 95%+ Recommended

Optimal Banner Selection by Situation

  • Gaming Apps: Use 320x50 or Adaptive as fixed bottom banner, insert 300x250 between levels
  • News/Feed Apps: Insert 300x250 between content, Adaptive Banner at bottom
  • Utility Apps: Fix 320x50 or 320x100 at bottom
  • Multi-country Services: Adaptive Banner required (handles device diversity)

The answer is 'the right combination for your app's UI and user behavior.' For maximum revenue, we recommend the following strategy:
